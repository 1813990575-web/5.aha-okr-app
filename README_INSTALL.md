# Aha OKR 安装指南

## 下载安装包

从 [GitHub Releases](https://github.com/1813990575-web/5.Aha-OKR/releases) 下载对应版本的 `.dmg` 文件：

- **Apple Silicon (M1/M2/M3)**：下载 `Aha OKR-x.x.x-arm64.dmg`
- **Intel Mac**：下载 `Aha OKR-x.x.x.dmg`

## 安装步骤

### 1. 打开 DMG 文件

双击下载的 `.dmg` 文件，会打开一个安装窗口：

```
┌─────────────────────────┐
│                         │
│    [Aha OKR 图标]       │
│                         │
│   拖拽此图标到右侧      │
│                         │
│        ↓ 拖拽到        │
│                         │
│    [Applications]       │
│        文件夹图标       │
│                         │
└─────────────────────────┘
```

### 2. 正确安装应用

**重要**：将左侧的 **Aha OKR 应用图标** 拖拽到右侧的 **Applications 文件夹图标** 中。

⚠️ **不要**直接把源码文件夹当成应用打开！

### 3. 首次运行

由于应用未经过 Apple 官方签名，首次打开时会显示安全警告。

#### 方法一：通过系统设置允许（推荐）

1. 首次打开应用时，会显示"无法验证开发者"警告
2. 点击"取消"
3. 打开 **系统设置** → **隐私与安全性**
4. 在"安全性"部分找到 "Aha OKR" 被阻止的提示
5. 点击 **"仍要打开"**

#### 方法二：使用终端（如果方法一无效）

如果应用显示"已损坏"，在终端执行：

```bash
xattr -d com.apple.quarantine /Applications/Aha\ OKR.app
```

### 4. 启动应用

安装完成后，可以从以下位置启动：
- **启动台**（Launchpad）
- **应用程序文件夹**（/Applications）
- **Spotlight 搜索**（Cmd + Space，输入 "Aha OKR"）

## 更新应用

当有新版本发布时：

1. 应用启动时会自动检查更新
2. 如果有新版本，会弹出提示对话框
3. 点击"去下载"跳转到 GitHub Releases 页面
4. 下载新版本并重复上述安装步骤（会覆盖旧版本）

## 卸载应用

1. 打开 **应用程序文件夹**
2. 找到 **Aha OKR**
3. 拖拽到 **废纸篓**，或右键选择"移到废纸篓"
4. 清空废纸篓

## 常见问题

### Q: 为什么显示"已损坏"？
A: 这是因为应用未经过 Apple 开发者签名。按照上述"方法二"移除隔离属性即可。

### Q: 如何区分真假应用图标？
A: 真应用图标：
- 位于 `/Applications` 文件夹
- 图标是一个独立的应用包
- 文件扩展名为 `.app`

假应用图标（源码文件夹）：
- 位于你的下载/工作目录
- 是一个普通文件夹
- 不应该直接打开

### Q: 如何查看应用版本？
A: 打开应用后，版本号显示在左下角。

- 支持 Apple Silicon 和 Intel 芯片
